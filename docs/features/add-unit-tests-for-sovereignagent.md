# Add Unit Tests for SovereignAgent
> Last updated: 2026-04-08
## Overview
This feature adds comprehensive unit tests for the `SovereignAgent` smart contract to validate its core functionalities. The tests cover critical functions and various edge cases to enhance reliability.
## How It Works
The tests are implemented in `src/__tests__/SovereignAgent.test.js` and `test/SovereignAgent.test.js`, focusing on the `postBounty`, `releaseBounty`, and `investSurplus` functions. Each test checks for expected outcomes based on different inputs and contract states.
## Configuration
No configuration required.
## Usage
Run the test suite using `npm test` to execute all unit tests and verify functionality.
