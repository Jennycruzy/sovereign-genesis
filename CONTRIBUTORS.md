# Contributors

Thank you to everyone who contributes to Sovereign-Genesis!

## How to Contribute

1. Fork this repository
2. Pick an open bounty from the [dashboard](http://130.162.217.77)
3. Complete the task and open a Pull Request
4. Include your wallet address in the PR description: `Wallet: 0xYourAddress`
5. The AI judge will review and release XTZ automatically if approved
